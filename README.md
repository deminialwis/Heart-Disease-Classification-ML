# Heart Disease Classification Using Machine Learning

## Project Overview

This project focuses on predicting the risk of heart disease using machine learning classification algorithms.

The main objective is to analyze patient health and demographic data and develop machine learning models that can classify whether a person is at risk of heart disease.

## Dataset

The dataset contains medical and lifestyle-related information about patients.

Features include:

* Age
* Gender
* Education level
* Smoking status
* Cigarettes per day
* Blood pressure
* Cholesterol level
* BMI
* Heart rate
* Glucose level
* Diabetes status
* Previous stroke history

### Target Variable

**Heart_Stroke**

* 0 - No heart disease risk
* 1 - Heart disease risk

## Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-learn
* Jupyter Notebook

## Machine Learning Algorithms

The following classification models were implemented:

1. Logistic Regression
2. K-Nearest Neighbors (KNN)
3. Random Forest Classifier
